# Implement a Component

You will be given a Figma design and your job is to implement the component in the eduriam-ui-core package.

## Input

- Figma design link
- Optionally description of how the component should work

## Workflow

1. ANALYZE carefully the Figma design and all of the used variables and structure.
2. IMPLEMENT the component precisely based on the Figma design.

## Details

- Always use the Figma MCP server to fetch the data about Figma design.
