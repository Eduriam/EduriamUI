# Project Overview

This project is a Yarn workspaces monorepo of React UI component libraries built with TypeScript and Material UI (MUI). Storybook is used for local development, documentation and testing of components.

The project consists of two main types of component libraries:

- **@eduriam-ui/core**
  - This is the core library of components built on Material UI components.
  - It contains components, interfaces, types and default theme settings.
- **@eduriam-ui/x**
  - This is a component library with more complex components.
  - This library builds on top of @eduriam-ui/core and also on Material UI
